Sobre

Este site surgiu a partir de uma atividade proposta pelo The Odin Project, com o objetivo de colocar em prática os conhecimentos adquiridos sobre HTML semântico. Além de estruturar o conteúdo de forma acessível e organizada, o projeto também visa a aplicação posterior de estilos com CSS, permitindo experimentar e evoluir na construção de interfaces web modernas e funcionais.
